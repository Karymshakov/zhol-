## 1. ОСУЩЕСТВИМОСТЬ РАБОТЫ

### 1.1 Технологическая база

#### Стек технологий (уже реализован)

| Слой | Технология | Назначение |
|------|-----------|------------|
| **Бэкенд** | Python 3.11 + FastAPI | REST API, маршрутизация, бизнес-логика |
| **База данных** | SQLite → PostgreSQL (prod) | Хранение профилей пользователей и результатов |
| **ORM** | SQLAlchemy 2.0 | Работа с БД через модели Python |
| **Аутентификация** | JWT-токены (bcrypt) | Безопасный вход и хранение сессий |
| **Фронтенд** | React 18 + TypeScript + Vite | Быстрый отзывчивый UI |
| **Стилизация** | TailwindCSS | Адаптивный дизайн (mobile-first) |
| **ИИ-сервис** | OpenAI GPT-4o-mini | Генерация персональных инсайтов и симуляций |
| **Деплой** | Любой VPS / Vercel + Railway | Облачное размещение |

#### Научно-методологическая база

- **RIASEC (Голланд)** — 6-типная модель профориентации (Realistic, Investigative, Artistic, Social, Enterprising, Conventional)
- **Ценности Шварца** — 7 базовых ценностей: автономность, влияние на мир, творчество, стабильность, признание, развитие, доход
- **Когнитивные стили Стернберга** — аналитический / интуитивный, системный / холистический мышление

### 1.2 Как платформа работает технически

```
Пользователь
    │
    ▼
[Тест 36 вопросов] ──► [React Frontend]
                              │
                              ▼
                    [FastAPI Backend /api]
                         │         │
                         ▼         ▼
                   [SQLite DB]  [GPT-4o-mini API]
                         │         │
                         └────┬────┘
                              ▼
                    [Профиль + RIASEC код]
                    [Топ-5 профессий КР]
                    [4 персональных инсайта]
                    [AI-симулятор рабочего дня]
```

**Алгоритм обработки:**
1. Пользователь отвечает на 36 вопросов (≈20 минут)
2. Система вычисляет баллы по 6 шкалам RIASEC, 7 ценностям и когнитивным стилям
3. Профиль сравнивается с базой из 48+ профессий через алгоритм совпадения
4. OpenAI GPT-4o-mini генерирует 4 персональных инсайта на русском с учётом реалий КР
5. Результат сохраняется в БД и доступен пользователю в любое время

---

### 1.3 Работа офлайн

**Текущее состояние:** платформа требует интернет-соединения для:
- генерации AI-инсайтов (OpenAI API)
- синхронизации результатов с облаком

**Офлайн-возможности (реализуемо):**
- **PWA (Progressive Web App)** — тест и статические данные кэшируются в браузере через Service Worker
- **Офлайн-тест** — 36 вопросов, алгоритм RIASEC и база профессий могут работать полностью локально (JavaScript, без сервера)
- **Локальные модели** — при необходимости полного офлайна: замена GPT-4o-mini на **Ollama + Mistral 7B** (работает на устройстве без интернета, ~4GB RAM)
- **Синхронизация при подключении** — результаты сохраняются локально и отправляются на сервер при появлении сети

### 1.4 Используемые ИИ-технологии

| Модель | Использование | Онлайн/Офлайн |
|--------|--------------|---------------|
| **OpenAI GPT-4o-mini** | Персональные инсайты, AI-симулятор рабочего дня | Онлайн (API) |
| **Mistral 7B (Ollama)** | Альтернатива для офлайн-режима | Офлайн (локально) |
| **Алгоритм RIASEC** | Сопоставление профиля с профессиями | Офлайн (код) |

**Почему GPT-4o-mini, а не GPT-4o?**
- В 10× дешевле при сопоставимом качестве для коротких текстов
- Достаточная точность для генерации инсайтов на русском
- Быстрый отклик (< 3 сек)

4. **Реклама** — партнёрство с вузами КР для рекламы специальностей целевой аудитории

**Рыночная ёмкость (КР):**
- 1 200+ первых участников уже на хакатоне
- Потенциал: 50 000+ активных пользователей в первый год при партнёрстве со школами

### 2.2 Ключевые инновации

#### 🔬 Инновация 1 — Мультимодельный анализ личности
Единственная платформа, которая одновременно применяет **три доказанные методики**:
- RIASEC → профессиональный тип
- Ценности Шварца → что важно в жизни
- Когнитивные стили Стернберга → как человек думает

Результат: не один тип «ты экстраверт», а многомерный профиль с пересечениями.

#### 🌐 Инновация 2 — Контекст Кыргызстана
Платформы типа Coursera или Proforientor.ru дают советы «в вакууме».
«Жол» знает:
- Средние зарплаты по профессиям в **КР** (а не в РФ или США)
- Список **вузов Кыргызстана** с соответствующими специальностями
- Проходные баллы **ОРТ** для поступления
- Особенности **рынка труда КР** (какие профессии востребованы реально)

#### 🎮 Инновация 3 — AI-симулятор рабочего дня
**Coursera** и **YouTube** показывают видео о профессии.
**«Жол»** позволяет **прожить рабочий день** в профессии:
- Реальные рабочие ситуации (придуманные GPT для конкретной профессии)
- Выборы с последствиями (влияет на Энергию, Стресс, Навыки, Настрой)
- Итоговый балл совпадения с профессией
- Персональные инсайты о том, как ты вёл себя в симуляции

#### 🤖 Инновация 4 — Персональный AI-советник на русском
После теста GPT-4o-mini генерирует **4 персональных инсайта** именно для этого пользователя:
- Учитывает конкретные баллы RIASEC и ценности
- Пишет о реалиях Кыргызстана
- Конкретные практичные советы, без шаблонных фраз

#### 📊 Инновация 5 — Визуализация профиля
Интерактивный отчёт включает:
- RIASEC-гистограмму с 6 шкалами
- Топ-5 профессий с процентом совпадения
- Карту ценностей
- Когнитивный стиль
- Конкретные следующие шаги

## 3. ИТОГОВЫЕ ВЫВОДЫ

**«Жол»** — это не учебная платформа (как Coursera) и не хранилище видео (как YouTube). Это **инструмент самопознания и принятия решений**, заточенный под конкретную страну, конкретную культуру и конкретного школьника.

**Три главных отличия:**
1. **Локальность** — данные КР, а не абстрактный мировой рынок
2. **Многомерность** — три методики + AI, а не один тест из 10 вопросов
3. **Действие** — симулятор позволяет попробовать профессию, а не просто прочитать о ней

**Готовность к масштабированию:**
- Платформа уже работает (MVP готов)
- 1 200+ участников — первая валидация
- Архитектура позволяет добавить Казахстан, Таджикистан и другие страны ЦА за 2–4 недели
